    class RedirectedStreams(stdout=None, stderr=None)¶

    Bases: object

    A context manager for capturing stdout and stderr output.

    >>> import uqbar.io
>>> import io

    >>> string_io = io.StringIO()
>>> with uqbar.io.RedirectedStreams(stdout=string_io):
...     print("hello, world!")
... 

    >>> result = string_io.getvalue()
>>> string_io.close()
>>> print(result)
hello, world!

    This context manager is not reentrant. Use a separate instance when nesting multiple timers.

    find_common_prefix(paths: Sequence[str | Path]) → Path | None¶

    Find the common prefix of two or more paths.

    >>> from pathlib import Path
>>> one = Path("foo/bar/baz")
>>> two = Path("foo/quux/biz")
>>> three = Path("foo/quux/wuux")

    >>> import uqbar.io
>>> str(uqbar.io.find_common_prefix([one, two, three]))
'foo'
