    class Table(children=None, *, attributes: Mapping[str, object] | Attributes | None = None, name: str | None = None)¶

    Bases: UniqueTreeList

    A Graphviz HTML table.

    >>> import uqbar.graphs
>>> table = uqbar.graphs.Table(
...     [
...         uqbar.graphs.TableRow(
...             [
...                 uqbar.graphs.TableCell("Cell One"),
...             ]
...         ),
...         uqbar.graphs.HRule(),
...         uqbar.graphs.TableRow(
...             [
...                 uqbar.graphs.TableCell("Cell Two"),
...             ]
...         ),
...     ]
... )
>>> print(format(table, "graphviz"))
<TABLE>
    <TR>
        <TD>Cell One</TD>
    </TR>
    <HR/>
    <TR>
        <TD>Cell Two</TD>
    </TR>
</TABLE>

    >>> table = uqbar.graphs.Table(
...     attributes={
...         "border": 5,
...         "bgcolor": "blue",
...     },
... )
>>> print(format(table, "graphviz"))
<TABLE BGCOLOR="blue" BORDER="5"></TABLE>
