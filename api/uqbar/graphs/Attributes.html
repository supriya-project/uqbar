Attributes¶

class Attributes(mode, **kwargs)[source]¶

An attributes listing for a Graphviz graph, cluster, node or edge.

>>> import uqbar.graphs
>>> attributes = uqbar.graphs.Attributes(
...     'node',
...     color='black',
...     fillcolor='white',
...     shape='Mrecord',
...     style=['rounded', 'filled'],
...     )

>>> print(format(attributes, 'graphviz'))
[color=black,
    fillcolor=white,
    shape=Mrecord,
    style="rounded, filled"];
