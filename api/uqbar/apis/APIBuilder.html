APIBuilder¶

class APIBuilder(initial_source_paths, target_directory, document_empty_modules=True, document_private_members=False, document_private_modules=False, member_documenter_classes=None, module_documenter_class=None, omit_root=False, root_documenter_class=None, title='API', logger_func=None)[source]¶

A builder of reStructuredText API documentation for Python packages.

>>> import uqbar
>>> import tempfile
>>> with tempfile.TemporaryDirectory() as target_directory:
...     with uqbar.io.DirectoryChange(target_directory):
...         builder = uqbar.apis.APIBuilder(
...             initial_source_paths=uqbar.apis.__path__,
...             target_directory=target_directory,
...         )
...         visited_paths = builder()
...

Parameters

  • initial_source_paths – a list of paths to scan for Python sources

  • target_directory – where to write reStructuredText output

  • document_private_members – whether to documenter private module members

  • document_private_modules – whether to document private modules

  • member_documenter_classes – a list of MemberDocumenter subclasses, defining what classes to use to identify and document module members

  • module_documenter_class – a ModuleDocumenter subclass

  • root_documenter_class – a RootDocumenter subclass
