Timer¶

class Timer(exit_message=None, enter_message=None, verbose=True)[source]¶

A context manager for timing blocks of code.

This context manager is not reentrant. Use a separate instance when nesting multiple timers.

Parameters

  • exit_message – message to print on entering the context

  • enter_message – message to print on exiting the context

  • verbose – whether to print output

>>> import math
>>> from uqbar.io import Timer
>>> timer = Timer("Elapsed time:", "Looping!")
>>> with timer:
...     for i in range(10000):
...         z = i ** math.pi
... 
Looping!
Elapsed time: 0.0019021034240722656
